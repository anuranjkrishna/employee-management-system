# Employee Management System

## 📌 Project Description
This is a Django-based Employee Management System.  
It helps to manage employee data like add, update, view and delete employees.

## 🚀 Features
- Add Employee
- Edit Employee Details
- Delete Employee
- View Employee List

## 🛠 Tech Stack
- Python
- Django
- HTML
- CSS
- SQLite
